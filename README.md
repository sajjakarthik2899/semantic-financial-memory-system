# semantic-financial-memory-system
LedgerPilot is an offline-first AI-powered semantic financial memory and analytics platform. Users can capture expenses in natural language, receive AI-driven corrections and categorization, store structured financial data locally and in the cloud, and gain insights through intelligent dashboards and future agentic AI assistance.

npm create vite@latest frontend -- --template react

why this?
1. This single terminal command is the modern "fast-track" way to set up a brand-new React project.
2. Instead of configuring dozens of tools manually, this command talks to npm (Node Package Manager), fetches a tool called Vite, and sets up a ready-to-go React development environment in a few seconds.
3. Vite includes a feature called Hot Module Replacement (HMR). When you change a piece of code in your React project, Vite instantly injects that change directly into the running Electron desktop window in milliseconds—without making you restart the app. It feels like editing a live document.
